Diseño Completo — ServerDash

Resumen para aprobación final antes de escribir el spec

STACK TECNOLÓGICO
Python + FastAPI Vue 3 + Vite SQLite JWT (access + refresh) HTTPS (cert autofirmado) Monaco Editor psutil · python-crontab
AUTENTICACIÓN & ROLES
admin
Acceso total. Gestión de usuarios, configuración del sistema.
operator
Ejecutar scripts, gestionar servicios, editar archivos.
readonly
Solo visualización. Sin modificaciones ni ejecuciones.
JWT access token (15min) + refresh token (7d) · HTTPS local → Let's Encrypt a futuro
LAYOUT & TEMA
Sidebar colapsable izquierdo + header con usuario/avatar · Tema Dark Pro (negro profundo + acentos azul/verde, estilo GitHub Dark) · Toggle dark ↔ light mode · Responsive para distintos tamaños de pantalla
MÓDULOS — 5 FASES
F1
Base + System Overview — infraestructura, auth, métricas CPU/RAM/disco/uptime, polling 5s
F2
Servicios systemd — listar, start/stop/restart, ver logs de journald
F3
Explorador de archivos — navegar, permisos, subir/bajar, mover/eliminar + Monaco Editor para editar archivos de texto y código
F4
Scripts favoritos — agregar .py .sh .cpp .cs, Monaco Editor para editar, ejecutar, ver output en quasi-realtime, historial de ejecuciones
F5
Crontab visual — calendar picker + selector de hora + frecuencia en lenguaje natural → genera cron expression. Listar, crear, editar, eliminar tareas.
MÓDULOS FUTUROS (plug-in)
Discos & SMART Red & puertos Usuarios Linux Docker containers Terminal web Notificaciones / alertas
PERSISTENCIA
SQLite — usuarios, roles, scripts favoritos, historial de ejecuciones · JSON config — preferencias del dashboard (tema, ajustes) · Sin dependencias externas · migrable a PostgreSQL a futuro
